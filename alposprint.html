🚀 Alpos Python

Compatible with Linux Mint 19.3 - 22.2

📋 Overview

Alpos Python is a printer utility package for Linux Mint systems that provides HTTP server functionality for printer management and text printing operations.

⚙️ System Requirements

Prerequisites

  • Operating System: Linux Mint 19.3 to 22.2 (64-bit)
  • Python: Python 3.x (included with Linux Mint)
  • Administrative Access: sudo privileges required for installation and configuration

📥 Installation Guide

1 Download the Package

Download the Debian package (.deb file) from the provided link:

👉 Click to Download alpos_python

Installation Methods:

GUI Method: Double-click the downloaded .deb file and follow the installation prompts

sudo dpkg -i alpos_python.deb sudo apt-get install -f

2 Verify Installation

sudo systemctl status http-server.service

Uninstallation

sudo apt-get remove alpospython

🔍 Printer Port Detection

First, list all connected USB devices:

lsusb

Test the printer ports to find which one is active. The command that successfully prints indicates your correct printer port:

ls > /dev/usb/lp0
ls > /dev/usb/lp1
ls > /dev/usb/lp2
ls > /dev/usb/lp3

Note: If a command prints successfully (you see output or the printer responds), that is your correct printer port. Usually it will be /dev/usb/lp0, /dev/usb/lp1, or /dev/usb/lp2.

⚙️ Printer Port Configuration

Important: Select your printer port from the dropdown below. The configuration command will update automatically with your selected port.

🖨️ Follow these steps to configure your printer port:

  1. Stop the HTTP server service:
    2. sudo systemctl stop http-server.service
  2. Edit the Python file with root privileges:
    3. sudo nano /usr/local/bin/http_server.py
  3. Find and change the target_path line:

    4. Look for this line in the file:

    target_path = '/dev/usb/lp0'

    Change it to your tested printer port (e.g., /dev/usb/lp1 or /dev/usb/lp2):

    target_path = '/dev/usb/lp2'

    Save the file (in nano: Ctrl+O, then Enter, then Ctrl+X to exit).

  4. Restart the HTTP server service:
    5. sudo systemctl start http-server.service
# Stop the service first sudo systemctl stop http-server.service # Edit the Python file sudo nano /usr/local/bin/http_server.py # After editing, restart the service sudo systemctl start http-server.service

Tip: You can also use this automated command to directly replace the target_path with your selected port:

sudo sed -i "s|target_path = .*|target_path = '/dev/usb/lp2'|" /usr/local/bin/http_server.py
